# Coaching Centre Management System

## Description
This project is a Coaching Centre Management System developed to manage student records, courses, and administrative tasks efficiently.

## Features
- Student registration
- Course management
- Record maintenance
- User-friendly interface

## Technologies Used
- Java / HTML / CSS / JavaScript
- MySQL

## How to Run
1. Download or clone the repository
2. Extract the zip file
3. Open the project in your IDE
4. Run the main file

- Source_Code/
